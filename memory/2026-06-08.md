# 2026-06-08 — Obsidian iCloud Bidirectional Sync (Completed)

## What Was Done

### Problem
- Wiki was at `~/.openclaw/wiki/main/` (local only)
- iCloud sync was one-way via cron job (every hour, was timing out)
- Phone edits in Obsidian wouldn't sync back to my workspace

### Solution: Option A — Move Wiki to iCloud Drive

**Steps:**
1. **Copied** wiki to iCloud Drive:
   ```
   ~/Library/Mobile Documents/com~apple~CloudDocs/OpenClaw Wiki/
   ```

2. **Replaced** `~/.openclaw/wiki/main` with symlink:
   ```
   ~/.openclaw/wiki/main -> ~/Library/Mobile Documents/.../OpenClaw Wiki
   ```

3. **Updated** `~/.openclaw/openclaw.json`:
   - Changed `vault.path` from `~/.openclaw/wiki/main` to `~/Library/Mobile Documents/com~apple~CloudDocs/OpenClaw Wiki`

4. **Disabled** old cron job:
   - Job ID: `8de4d3d8-e0aa-434c-8eda-98089bfef7d0`
   - Was timing out (27 consecutive errors)

5. **Restarted** gateway to pick up new config

### Result
- **Bidirectional sync works**
- My writes → iCloud → phone: Instant
- Phone writes → iCloud → me: Instant
- No more hourly delay or cron timeouts

### Verified

- Test file written via symlink appeared in iCloud
- Test file written in iCloud appeared via symlink
