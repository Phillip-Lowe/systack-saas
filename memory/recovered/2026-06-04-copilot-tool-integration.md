2. **If stuck** → open Copilot chat
3. **Capture response** → analyze for usefulness
4. **Save to memory** → document insight + source
5. **Apply to current task** → integrate learning

### Memory Storage Targets
- `MEMORY.md` → key insights, decisions, lessons
- `memory/YYYY-MM-DD.md` → raw consultation logs
- `TOOLS.md` → updated access info, new findings
- Wiki → structured knowledge if applicable
