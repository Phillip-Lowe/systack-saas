## Copilot's Self-Generated Prompt

```
A cinematic hero image of a powerful autonomous AI personal agent operating in the background of a modern business workflow.

Scene: A clean, minimal digital workspace blending physical and digital reality — a desk with a laptop and phone in the foreground, while a transparent holographic interface expands outward showing connected systems: calendar scheduling, payment processing, customer bookings, and data flows all happening automatically.

The agent is not a character but a presence — represented as a subtle glowing intelligence (soft light core or abstract neural energy) orchestrating everything seamlessly.

Visual elements:
- Real-time booking confirmation panels
- Payment flow (Square-style checkout visuals)
- Calendar slots auto-filling
- Data streams flowing between systems
- UI elements floating and connecting intelligently

Style:
- Ultra-modern, high-end SaaS aesthetic
- Clean, minimal, no clutter
- Slightly cinematic lighting (cool blue, neutral tones, subtle glow accents)
- Photoreal + futuristic UI overlay hybrid
- Depth of field, soft focus edges, premium branding feel

Mood:
- Effortless automation
- Quiet control
- Speed and precision
- Zero friction
- "It just works"

Composition:
- Foreground = simple human workspace
- Midground = active interface
- Background = abstract connected system network

Avoid:
- Cartoony robots
- Overly sci-fi neon chaos
- Busy cluttered dashboards

Goal: Visually communicate that this AI agent removes all friction and runs the business automatically in seconds.
```

## Key Lessons

1. **Copilot chat cannot browse websites** — always provide context in the prompt or use web search first
2. **Copilot's self-generated prompts can be excellent** — when given general context, it creates detailed, structured prompts with clear sections (Visual elements, Style, Mood, Composition, Avoid, Goal)
3. **Multiple tools produce different results from same prompt** — Copilot vs Kling will interpret differently
4. **Prompt structure matters** — Copilot's cinematic/structured approach produced a more premium-feeling result
5. **For future image generation:** Consider using Copilot to *refine* the prompt, then feeding that refined prompt to other tools (Kling, local image gen, etc.)
