# MOD 1 — Premium Wearable Artifacts Brand

## Identity

**MOD 1** is a premium apparel system built on narrative-driven, high-concept design.

*   **Positioning:** Not clothing → it is **wearable artifacts / prototypes**.
*   **Tagline:** *Wear the Prototype*
*   **AI stance:** AI is an internal tool, not a marketing theme.
*   **Core rule:** Every piece = **self-contained universe**. No reused concepts, no shared lore unless explicitly declared.

## Product Architecture

```
Prototype → Collection Layer
 └── Mold → Individual design
 └── Product Variants → hoodie / tee / joggers / etc.
```

*   **Prototype:** Unique theme, system, or philosophical lens.
*   **Mold:** Specific moment, event, or visual narrative.
*   **Product:** Artifact extracted from that moment.

## Visual System Rules

*   **Export:** PNG 4200×5400, 300 DPI, transparent.
*   **Composition:** Foreground / midground / background logic.
*   **Color Roles:** Base, Complementary, Analogous, Neutral.
*   **Style:** Cinematic, photoreal or hyper-detailed. No AI sheen. No over-sharpening. Controlled color usage.

## Model / Media System

*   **Audience:** Male 18–35 (up to 45).
*   **Style:** Real-life candid, no camera acknowledgment, clean faces, natural lighting or soft editorial.
*   **Video Tool:** Tolstoy (shifted from Runway).
*   **Focus:** Clean motion, no body distortion, minimal studio environment. Emphasis on fit, texture, branding.

## Operating Philosophy

1.  **Non-repetition:** Every output must feel new.
2.  **Artifact Thinking:** You are not making clothes; you are releasing **objects with meaning**.
3.  **Controlled Chaos:** Creative freedom, but inside structured systems.
4.  **System Memory:** Everything is reusable — prompts, templates, assets, workflows.
