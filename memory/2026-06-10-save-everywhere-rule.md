# Save Everywhere Rule

## Directive (2026-06-10 05:17 CDT)
When user says "save this everywhere" or similar, save to ALL available locations:

1. Daily memory: `memory/YYYY-MM-DD-descriptive-name.md`
2. Curated memory: `MEMORY.md` (add to relevant section)
3. Wiki: `.openclaw/wiki/main/Page-Name.md`

Never ask "where do you want it saved." Save everywhere by default.
