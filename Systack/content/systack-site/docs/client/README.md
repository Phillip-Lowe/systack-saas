# Client Document Portal

This directory contains client-facing documents hosted at:
`https://systack.net/docs/client/`

## Documents

| Document | File | Purpose |
|----------|------|---------|
| Service Manual | `SyStack-Service-Manual-Client-v1.0.pdf` | Full service overview |
| Deployment Guide | `SyStack-Enterprise-Deployment-Guide-v1.0.pdf` | Enterprise setup instructions |

## Updating Documents

1. Edit the source `.md` file in `/documents/guides/`
2. Generate PDF using the branded PDF generator
3. Copy PDF to this directory
4. Update links in customer dashboard
